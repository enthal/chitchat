# chitchat
A simple chat server made with angularjs, socket.io, coffeescript, nodejs, redis, and ‡•ê.

## Up and running

Install [node.js](http://nodejs.org/download/)
and [npm](http://howtonode.org/introduction-to-npm)
and [coffeescript](http://coffeescript.org/).

```sh
git clone git@github.com:enthal/chitchat
cd chitchat
npm install
./server.coffee
```
